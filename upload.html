CSV Requirements

Required columns

Column Type Description
rating_overall Number (1–5) Overall rating given in the review.
like_count Integer (0+) Number of likes / upvotes for the review.
city Text City where the restaurant is located.
restaurant_name Text Name of the restaurant.
created_at Datetime Review timestamp in YYYY-MM-DD or YYYY-MM-DD HH:MM:SS.
primary_cuisine Text Main cuisine label (e.g., “Italian”, “North Indian”).
review_text Text Full written review from the customer.
restaurant_review_count Integer Total reviews for that restaurant on the source platform.

General instructions

  • File must be CSV with a header row.
  • One row represents a single review.
  • Column names should match exactly (case‑sensitive).
  • Missing values are allowed but may reduce detail in some sections.
  • If only one city, one restaurant, or very few cuisines exist, some comparison views in the report will show as “N/A”.
